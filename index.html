HTML5

HTML5 (HyperText Markup Language, versión 5) es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML. HTML5 especifica dos variantes de sintaxis para HTML: una «clásica», HTML (text/html), conocida como HTML5, y una variante XHTML conocida como sintaxis XHTML5 que deberá ser servida como XML (application/xhtml+xml).

HTML5 introduce nuevos elementos y atributos que reflejan el uso típico de los sitios web modernos. Algunos de ellos son técnicamente similares a las etiquetas <div> y <span>, pero tienen un significado semántico, como <nav> (bloques de navegación) y <footer>.

CSS3

CSS3 (Cascading Style Sheets, nivel 3) es la última evolución del lenguaje de hojas de estilo en cascada. CSS3 trae consigo muchas novedades esperadas, como las esquinas redondeadas, sombras, gradientes, transiciones o animaciones, además de nuevos diseños como multi-columnas, cajas flexibles o diseños de cuadrícula (grid layouts).

Las hojas de estilo en cascada permiten separar el contenido de la presentación, facilitando el mantenimiento y la accesibilidad de los sitios web. Con CSS3 se pueden crear diseños responsive que se adaptan a diferentes dispositivos y tamaños de pantalla.

JavaScript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico. Se utiliza principalmente en su forma del lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas.

JavaScript se ha convertido en una tecnología fundamental para el desarrollo web moderno. Permite crear aplicaciones interactivas, validar formularios, manipular el DOM, realizar peticiones asíncronas (AJAX), y mucho más. Con frameworks como React, Angular y Vue.js, JavaScript ha revolucionado el desarrollo frontend.
