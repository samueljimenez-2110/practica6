Sobre este Proyecto

Este sitio web fue desarrollado como parte de la Práctica 7 del curso de Programación de Aplicaciones Web. El objetivo principal es demostrar el conocimiento adquirido en el uso de HTML5 y CSS3 para crear interfaces web atractivas y funcionales.

Durante el desarrollo de este proyecto, se implementaron las siguientes tecnologías y técnicas:

  • Estructuración semántica con HTML5
  • Estilización avanzada con CSS3
  • Diseño de navegación intuitiva
  • Implementación de efectos hover y transiciones
  • Creación de diseño responsive
  • Uso de selectores CSS específicos

Objetivos de Aprendizaje

A través de esta práctica, se lograron los siguientes objetivos:

  • Comprender la separación entre estructura (HTML) y presentación (CSS)
  • Implementar hojas de estilo externas vinculadas correctamente
  • Aplicar selectores CSS para estilizar elementos específicos
  • Crear menús de navegación interactivos con efectos visuales
  • Mantener consistencia de diseño en múltiples páginas
  • Desarrollar interfaces web profesionales y atractivas
